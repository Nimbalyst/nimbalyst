# /track-idea Command

Shortcut for `/track idea [description]`.

Track an idea: $ARGUMENTS

Add to `nimbalyst-local/tracker/ideas.md` with format:
```markdown
- [Description] #idea[id:idea_[ulid] status:new priority:medium created:YYYY-MM-DD]
```

Generate ULID, set status to `new`, and confirm tracking.
