# /track-bug Command

Shortcut for `/track bug [description]`.

Track a bug: $ARGUMENTS

Add to `nimbalyst-local/tracker/bugs.md` with format:
```markdown
- [Description] #bug[id:bug_[ulid] status:to-do priority:medium created:YYYY-MM-DD]
```

Generate ULID, determine priority from keywords, and confirm tracking.
