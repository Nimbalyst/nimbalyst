# Nimbalyst v0.46.1 Release Notes

### New Features

- **Model version display**: Claude Code model versions now shown in the model selector for better visibility into which AI model you're using
- **Prompt caching support**: AI responses now benefit from prompt caching, reducing API costs and improving response times for repeated queries

### Improvements

- **Session timestamps**: Session timestamps now match sort order for more consistent and intuitive display throughout the application

### Fixed

- **Model migration**: Users migrating from older versions now correctly see all available Claude Code models in the selector
